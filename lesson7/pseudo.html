селектор::псевдо-элемент {
    свойство: значение;
}

1. Первая буква элемента:

    color: red;
    text-transform: uppercase;
    font-weight: bold;

4. Before / After

before - псевдоэлемент, который является первым потомком выбранного элемента (по умолчанию строчный)

after - псевдоэлемент, который является последним потомком выбранного элемента (по умолчанию строчный) 

.before-after::before {
    content: "♥";
    color: red;
    margin-right: 5px;
}

.before-after::after {
    content: "\2B50";
    margin-left: 5px;
}

Псевдоклассы

Псевдокласс CSS - это ключевое слово, добавленное к селектору, которое определяет его особое состояние, например, нажатая ссылка 

селектор:псевдо-класс {
    свойство: значение;
}

1. Дочерние элементы:

  • селектор:first-child - первый дочерний элемент
  • селектор:last-child - последний дочерний элемент
  • селектор:nth-child(число) - например, :nth-child(2) - дочерний элемент под номером 2
  • селектор:nth-child(odd) - все нечетные дочерние элементы
  • селектор:nth-child(even) - все четные дочерние элементы
nav a {
    border-left: 3px solid salmon;
}

nav a:first-child {
    border-left: none;
}

4. Псевдоклассы элементов формы:

  • селектор:focus {стили} - применяется, когда элемент (такой как input) получает фокус
  • селектор:checked {стили} - input с типами radio и checkbox, option, которые выбраны
  • селектор:default {стили} - элемент формы, установленный по умолчанию для группы связанных элементов (radio или checkbox)
  • селектор:disabled {стили} - элемент, с установленным атрибутом disabled
  • селектор:enabled {стили} - элемент, без атрибута disabled
  • селектор:optional {стили} - элементы, для которых не был установлен атрибут required
  • селектор:required {стили} - элементы, для которых был установлен атрибут required
  • селектор:valid {стили} - поля, которые прошли html валидацию
  • селектор:invalid {стили} - поля, которые не прошли html валидацию
  • селектор:in-range {стили} - элементы, значение которых соответствует указанному для них диапазону ограничений
  • селектор:out-of-range {стили} - элементы, значение которых не соответствует указанному для них диапазону ограничений
