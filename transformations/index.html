Transition (переходы)

изменение времени перехода элемента из одного состояния в другое.
  1. переходы (transition) применяются ко всем элементам, а также к псевдоэлементам ::before и ::after
  2. не все CSS свойства можно анимировать (например, свойство display не доступно для анимации)
  3. список доступных для анимации свойств

Переходы задаются следующими свойствами:

  1. transition-property - свойство перехода, которое нужно анимировать. Если необходимо анимировать несколько свойств, они перечисляются через запятую. В значении 'all' будут анимированны все свойства
  2. transition-duration - скорость перехода свойства из одного состояния в другое. Можно задать в секундах (s) или миллисекундах (ms)
  3. transition-delay - время, после которого свойство начнет менять свое состояние. Можно задать в секундах (s) или миллисекундах (ms)
  4. transition-timing-function - функция, которая описывает скорость перехода от одного значения к другому
  5. transition - универсальное свойство, для описания предыдущих: 
    transition: transition-property transition-duration transition-timing-function transition-delay;
При наведении элемент становится непрозрачным за 2 секунды 
.transition-opacity {
    opacity: .5;
}

.transition-opacity:hover {
    opacity: 1;
    transition: opacity 2s linear;
}
При наведении элемент меняет цвет за 1 секунду и цвет фона за 2 
.transition-color {
    color: rgb(24, 255, 255);
    background-color: rgb(0, 184, 212);
    transition: color 1s linear, background-color 3s ease;
}

.transition-color:hover {
    background-color: rgb(24, 255, 255);
    color: rgb(0, 184, 212);
}

Transform (трансформации)

позволяют сдвигать, поворачивать и масштабировать элементы. Трансформации преобразовывают элемент, не затрагивая остальные элементы веб-страницы, т.е. другие элементы не сдвигаются относительно него.

Трансформации задаются следующими свойствами:

  1. translate - смещает элемент вниз/вверх, влево/вправо. 
    translate(x,y) - смещение по оси X и по оси Y
    translateX(n) - смещение по оси X
    translateY(n) - смещение по оси Y
  2. scale - масштабирует элементы, делая их больше или меньше. Значения от 0 до 1 уменьшают элемент. 
    scale(x,y) - первое значение масштабирует элемент по ширине, второе — по высоте
    scaleX(n) - масштабирует элемент по ширине
    scaleY(n) - масштабирует элемент по высоте
  3. rotate - поворачивает элементы на заданное количество градусов, отрицательные значения (от -1deg до -360deg) поворачивают элемент против часовой стрелки, положительные — по часовой стрелке. 
    rotate(45deg) поворачивает элемент на 45 градусов по часовой стрелке
  4. skew - используется для искажения сторон элемента относительно координатных осей. Если указано одно значение, второе будет определено браузером автоматически. 
    skew(x-угол,y-угол) - искажение по x и по y
    skewX(угол) - искажение относительно оси X
    skewY(угол) - искажение относительно оси Y
При наведении элемент смещается по оси Y 
.translate:hover {
    transform: translateX(3rem);
}
При наведении элемент увеличивается в ширину и в высоту 
.scale:hover {
    transform: scale(1.2, 1.5);
}
Элемент повернуты на 5 градусов против часовой стрелки 
.rotate {
    transform: rotate(-5deg);
}
Искажение относительно X на 10 градусов 
.skew {
    transform: skewX(10deg);
}

Animation (анимация)

придают динамичность страницам.
  1. анимации применяются ко всем элементам, а также к псевдоэлементам ::before и ::after
  2. не все CSS свойства можно анимировать (например, свойство display не доступно для анимации)
  3. список доступных для анимации свойств

Объявление анимации

@keyframes название-анимации {
    набор правил:
    0% { стили } — первый шаг анимации
    50% { стили } — шаг на половине анимации
    100% { стили } — последний шаг анимации
}

Анимации задаются следующими свойствами:

  1. animation-name - название анимации
  2. animation-duration - как долго длится анимация (в секундах - s или миллисекундах - ms)
  3. animation-timing-function: как вычисляются промежуточные состояния
  4. animation-delay: анимация начинается спустя некоторое время
  5. animation-iteration-count: сколько раз должна выполняться анимация (infinite - бесконечно)
  6. animation-direction: должно движение идти в обратную сторону или нет
    1. normal - идёт с самого начала, после завершения цикла возвращается к исходному состоянию
    2. alternate - идёт с начала до конца, затем плавно возвращается в исходное положение
    3. reverse - идёт с конца цикла, после его завершения возвращается к исходному состоянию
    4. alternate-reverse - идёт с конца до начала, затем плавно возвращается в исходное положение
  7. animation-fill-mode: какие стили применяются до начала анимации и после её завершения
    1. none - не применяются какие-либо стили
    2. forwards - к элементу по окончанию анимации применяется стиль последнего кадра
    3. backwards - к элементу по окончанию анимации применяется стиль первого кадра
    4. both - к элементу по окончанию анимации элементу применяются оба правила (как для forwards, так и для backwards)
Бесконечное изменение прозрачности элемента 
@keyframes blink {
    0% {
        opacity: 0;
    }
    50% {
        opacity: .5;
    }
    100% {
        opacity: 1;
    }
}

.animation-blink {
    animation: blink 5s ease 0s infinite alternate forwards;
}
Бесконечное изменение размера и цвета элемента 
@keyframes color-size {
    0% {
        background-color: rgb(96, 125, 139);
        width: 300px;
    }
    25% {
        background-color: rgb(33, 150, 243);
        width: 320px;
    }
    50% {
        background-color: rgb(76, 175, 80);
        width: 340px;
    }
    75% {
        background-color: rgb(255, 152, 0);
        width: 360px;
    }
    100% {
        background-color: rgb(244, 67, 54);
        width: 380px;
    }
}

.animation-color-size {
    padding: 1rem;
    animation: color-size 5s ease 0s infinite alternate forwards;
}

Box-shadow (тени)

внешние и внутренние.

Тени задаются следующими свойствами: 
box-shadow: смещение-по-ширине смещение-по-высоте радиус-размытия размер-тени цвет-тени;
box-shadow: inset смещение-по-ширине смещение-по-высоте радиус-размытия размер-тени цвет-тени;

  • смещение-по-ширине - положительное значение рисует тень, смещенную вправо от текста, отрицательная длина — влево
  • смещение-по-высоте - положительное значение смещает тень вниз, отрицательное — вверх
  • радиус-размытия - если значение размытия равно нулю, то край тени четкий, в противном случае, чем больше значение, тем больше размыт край тени. Отрицательные значения не допускаются
  • размер-тени - расстояние, на которое тень увеличивается. Положительные значения заставляют тень расширяться во всех направлениях на указанный радиус. Отрицательные значения заставляют тень сжиматься
  • цвет тени - черный, если не задан другой
  • inset - тень блока отображается с внутренней стороны
Внешняя тень 
.shadow {
    box-shadow: 3px 4px 8px 6px #FF4E00;
}
Внутренняя тень 
.inset-shadow {
    box-shadow: inset 0 0 6px 6px #FF4E00;
}
